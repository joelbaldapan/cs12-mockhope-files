# cs12-mockhope-files
Files for Joel Baldapan's 1st CS12 DOOM: Dry-run Of Our Misery

## Instructions
1. Clone the repository to your local files:
```bash
git clone https://github.com/joelbaldapan/cs12-mockhope-files.git
```
2. Navigate to the newly created directory:
```bash
cd cs12-mockhope-files
```
3. Choose which project to implement, based on the specifications on the [DOOM 1 Website](https://joelbaldapan.github.io/cs12-mockhope/):
```
.
├── koyuki_the_game
└── ultimate_ttt
```
4. Implement them! **Good luck! :D**
